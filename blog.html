1. What is the DOM?

The DOM is a programming interface that represents the structure of a document (like an HTML or XML file) as a tree of objects. Each node in the tree corresponds to a part of the document, such as elements, attributes, and text.


2. How do you select an element from DOM?

Selecting an Element from the DOM:

  • document.getElementById('id')
  • document.querySelector('.class')
  • document.getElementsByClassName ('class')
  • document.getElementsByTagName ('tag')


3. What is event delegation in the context of DOM, and why is it useful?

Event delegation is a technique that allows you to handle events at a higher level in the DOM rather than attaching event handlers to individual elements. It's useful for optimizing performance and managing dynamic content because it reduces the number of event listeners needed and ensures that new elements can automatically inherit the event handler.


4. How do you manipulate an element's attributes and styles using the DOM?

You can manipulate attributes and styles using:

  • Attributes: element.setAttribute('attr', 'value') or element.getAttribute('attr')
  • Styles: element.style.property = 'value' (e.g., element.style.color = 'red') or by adding/removing CSS classes with element.classList.add ('class') or element.classList.remove('class')
